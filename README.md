# 📦 PSF Inventario - Dashboard de Productos Terminados

Dashboard web automático para visualizar productos terminados en tiempo real.

## 🎯 Características

- ✅ Dashboard web responsive
- ✅ Lectura automática cada mañana a las 6 AM
- ✅ Filtros y búsqueda en tiempo real
- ✅ Sin licencias requeridas
- ✅ Colores PSF

## 📋 Setup

1. Clonar este repo
2. Crear cuenta Supabase
3. Registrar app en Azure
4. Deploy a Vercel
5. Configurar Cron job

Ver SETUP.md para instrucciones detalladas.
